Portfolio Website
This is a Next.js portfolio website converted from your resume.

Getting Started
Since the project files were generated manually, you need to install the dependencies first.

Open your terminal and navigate to the project directory:

cd portfolio
Install dependencies:

npm install
Run the development server:

npm run dev
Open http://localhost:3000 with your browser to see the result.

Project Structure
app/page.tsx: The main page containing all the portfolio content.
components/ui: Reusable UI components like SectionCard, SkillBadge, etc.
app/globals.css: Global styles and Tailwind directives.
tailwind.config.ts: Tailwind CSS configuration.
Technologies Used
Next.js 14 (App Router)
Tailwind CSS
Framer Motion (for animations)
Lucide React (for icons)
